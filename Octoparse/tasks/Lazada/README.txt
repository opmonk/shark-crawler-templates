Octoparse have a bug where pagination is not getting the correct data on Lazada Listings.
This is a hybrid approch to address the octoparse issue
 
1. To get the URL's install browser extension IPSharkPagesGenerator
2. Click on the icon then select Platform then click Crawl
3. Paste the URL's generated by the browser extension to Octoparse
4. Run Locally on Octoparse
5. Then Run python process_lazada.py -p LazadaID -f "C:/Users/chris/Desktop/octoparse_data/LazadaID" -u 'https://www.lazada.co.id'
6. It will create a folder with date as name then upload to S3
